폰트를 두껍게

<strong>태그는 semantic tag로 html문서 내에서 강조하고 싶은 내용에 사용한다. 둘 다 폰트가 두꺼워지지만 이건 내용강조

<i>태그는 css처럼 웹사이트에 표시 되는 모양만 결정(스타일링) 이탤릭체 내용강조없이 모양만 이탤릭체로 표현하고 싶을 때는 css font-style 속성을 사용한다.

<em>태그는 semantic tag로 html에서 이탤릭체로 내용을 강조하고 싶을 때 사용한다.

<s>태그는 화면상에 표시되는 모양 취소선 취소된 내용이라는 강조가 아닌 디자인적으로 중간에 가로선을 긋고 싶을떄 css text-decoration 속성을 사용한다.

<del>태그는 semantic tag로 문서 내에서 삭제된 내용이라는 걸 강조하고 싶응 떄 사용한다.
